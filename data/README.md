# data/

Locally acquired corpora live here (this directory is gitignored except for
this file). Expected layout: one directory per corpus under `data/raw/`, e.g.
`data/raw/dialektarium/manifest.csv`.

See [docs/datasets.md](../docs/datasets.md) for per-corpus acquisition
instructions and the manifest format, and run `tulip data list` to check
what tulip can find.
